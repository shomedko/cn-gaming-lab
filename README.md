# cn-gaming-lab

一个用于归档与展示独立 HTML 页面的仓库，所有页面均为单文件 HTML 结构，不依赖外部域名或特定网站环境。

## 项目简介

本仓库旨在收集和整理一系列独立 HTML 页面，涵盖游戏相关工具、交互演示、信息展示等类型。每个页面均以单一 HTML 文件形式存储，便于直接查看、分享或离线使用。

## 目录结构

```
cn-gaming-lab/
├── index.html          # 仓库首页 / 页面导航
├── pages/              # 存放独立 HTML 页面
│   ├── game-tool.html
│   ├── info-card.html
│   └── ...
└── resources/          # 页面共享资源（可选）
    └── ...
```

- `index.html`：仓库首页，提供页面索引与基本导航。
- `pages/`：所有独立 HTML 页面存放于此，文件名体现页面主题。
- `resources/`：若页面需共用样式、脚本或图片，可存放于此（非必需）。

## 页面归档说明

- 每个页面均为独立 HTML 文件，可直接在浏览器中打开运行。
- 页面之间无交叉依赖，不依赖外部服务器或数据库。
- 部分页面可能包含内嵌样式与脚本，无需额外加载外部资源。
- 页面内容保持简洁、稳定，不包含动态外部链接。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面。
- 新增页面请放入 `pages/` 目录，并确保文件为纯 HTML 格式。
- 如需引用共享资源，请将其放入 `resources/` 目录，并在页面内使用相对路径。
- 不建议在页面内嵌入外部域名链接或追踪代码。
- 本仓库不针对任何特定游戏、网站或商业服务，仅作为个人或社区项目归档。

## 使用方式

1. 克隆或下载本仓库。
2. 打开 `index.html` 查看页面索引。
3. 点击页面列表中的链接，或直接打开 `pages/` 下的 HTML 文件。
4. 所有页面均可离线浏览。
